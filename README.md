# 📊 Customer Shopping Analysis

## 🔹 Project Overview
This project focuses on analyzing customer shopping behavior using an end-to-end data analytics approach. It includes data cleaning using Python, data analysis using SQL, and dashboard visualization using Power BI to generate meaningful business insights.

---

## 🔹 Tools & Technologies Used
- Python (Pandas)
- SQL (Data Analysis)
- Power BI (Dashboard & Visualization)
- CSV Dataset

---

## 🔹 Data Cleaning using Python
- Checked dataset structure using .info() and .describe()
- Handled missing values and removed duplicates
- Standardized column names
- Converted date columns to proper datetime format
- Created new derived columns for analysis
- Exported cleaned dataset for further use

---

## 🔹 Data Analysis using SQL
- Queried dataset using SELECT statements
- Filtered data using WHERE clause
- Used DISTINCT to handle duplicate records
- Created calculated fields using CASE statements
- Performed aggregations using GROUP BY
- Extracted insights for dashboard creation

---

## 🔹 Dashboard (Power BI)
- Built an interactive dashboard to visualize:
  - Sales Trends
  - Customer Segmentation
  - Category-wise Performance
  - Subscription Analysis
